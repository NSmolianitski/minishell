# Minishell
21-school project. Little shell with commands (echo, cd, pwd, export, unset, env, exit) and some more stuff

## План

### Функции:

- echo
- cd
- ~~pwd~~
- export
- unset
- env
- exit

### Другое:

- ;
- кавычки
- редиректы (< > << >>)
- пайпы (|)
- переменные окружения
- $? (вывод return'a последней выполненной команды)
- сигналы (ctrl + C, ctrl + D, ctrl + \)
